## 1.2 Android App


- Install from [Google Play](https://play.google.com/store/apps/details?id=com.sameerasw.airsync) ( [web](https://play.google.com/apps/testing/com.sameerasw.airsync) )
- Or [GitHub releases](https://github.com/sameerasw/airsync-android/releases/latest)
    

### Sideloading instructions

1. Download the latest app-release.apk from GitHub releases.
    
    https://github.com/sameerasw/airsync-android/releases/latest
    
2. Install the downloaded apk file. 
    
    <aside>
    ⚠️
    
    Installation might get blocked by play protect due to the app using sensitive permissions and being sideloaded. If there is no visible option to skip the check, you may need to go to Google Play Store > Account menu > Play Protect and disable temporarily during the installation.
