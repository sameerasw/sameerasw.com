## System Requirements

- Both apps should be in the same local area network such as WiFi and Ethernet.
- VPN, Proxy connections may prevent device discovery.
- In some rare occasions, Internet solutions such as netsticks might not put the mac in the same local area network as other devices over WiFi.

### Android App

Minimum Android version : Android 11 (SDK 30)

### mac App

Minimum macOS version : Sonoma - macOS 14.5
