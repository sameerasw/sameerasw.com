### 4.1.1 macOS App

- Incompatible macOS versions but, this will be causing app not to open.
- There really shouldn’t be any issues during installation. Please check if the downloaded dmg file is corrupted.

### 4.1.2 Android App

- Installation can be blocked if (Most of these issues are no longer applicable as we are installing through the google play)
    - The browser used to download the .apk file does not have the permission to install apps from unknown sources
    - Play protect being enabled during the installation.
    - OEM specific sideloading prevention features like Advanced/ Enhanced protection mode in Pixels and Auto Blocker in Samsung OneUI can prevent the app from installing, Temporarily disable them in order to continue.
    - Error while parsing the package might mean you are on an outdated version of Android.
