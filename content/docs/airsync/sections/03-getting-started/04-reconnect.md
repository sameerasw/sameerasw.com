## 2.4 Re-connect

- Auto-reconnect is on by default. It will try to reconnect to the last device if it was disconnected unexpectedly. You can cancel re-connection if you’re gonna be away for a while.
- Also you can use the QS tile to quickly re-connect back to the last device on this network.

- Nearby devices sharing the same network will show up in the mac app allowing you to easily reconnect if device discovery is enabled.
