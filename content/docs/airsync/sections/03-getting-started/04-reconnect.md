## 2.4 Re-connect

- Auto-reconnect is on by default. It will try to reconnect to the last device if it was disconnected unexpectedly once every 10 seconds for a minute and then once a minute. You can cancel re-connection if you’re gonna be away for a while.
- Also you can use the above QS tile to quickly re-connect back to the last device on this network.
