# What's New
- Essentials is now verified by the new Android Developer Verification program. Hope this helps with installation with upcoming sideloading restrictions.
    <img width="300" alt="image" src="https://github.com/user-attachments/assets/593d3af2-0680-4caa-98cd-88874fc77701" />

- New notification lighting effect
- QS tile categories
- Shape effects for album art in ambient music glance
- Many improvements to ambient music glance and fixes
- Update note onboarding
- You can use usage access permission instead of Accessibility for 
   - Freeze
   - App lock
   - Dynamic nightlight
   features. Enable in settings. 
- A lot of translations
- And a ton of minor improvements and bug fixes
