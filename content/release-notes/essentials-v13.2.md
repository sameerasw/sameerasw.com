# What's New?
- Custom refresh rate controls with QS tile
- App lock timeout
- Live wallpaper with custom videos
- Marquee effect for UI text in larger scale WIP
- Circle to search custom gesture now does not require gesture bar to be hidden
- Circle to search gesture height adjustment

## Fixes
- Volume buttons for call audio when using remap
- Flashlight live update notification getting stuck
- Device becomes unresponsive while using Ambient Glance in hub mode with Android Auto
- Shizuku permission label after granting permission
- And more
