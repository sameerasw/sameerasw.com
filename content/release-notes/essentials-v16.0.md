# What's New?
- Power and battery
   - Custom trigger level (Below 20%)
   - Customize what to turn off (AOD, Dark mode, Brightness, Doze ... )
- Notification snoozing
   - Enable snoozing
   - Set a custom default timer
   - Set custom timer options
- DNS cycle mode in QS tile
   - Instead of toggling, now you can cycle between your added DNS presets by tapping the QS tile.
   - Long press the tile to customize
- Charging optimization tile customization
   - Now you can make the tile toggle between Adaptive and Limit skipping not optimized or whatever you prefer
   - Long press to customize
- Disable loud volume dialog
- Are we there yet
   - Are we there yet wearOS toggle
   - Fix abnormal power draw of watch during Are we there yet
   - Tapping the notification now opens the navigation UI
   - Navigation UI now offers a button to view all location settings
   - Fixed live update appearing after ending a trip
- Fix searchbar crash
- WearOS sound mode now only syncs one way to watch
- More lottie animations (Getting used to Figma Motion =D )
- SystemUI restart QS tile
- Add more fixes for progressive blur use across the app
- Fix app lock permission usage causing the overlay to fail
- Fix certain feature cards triggering feature highlight outside search
- And  more fixes and improvements
