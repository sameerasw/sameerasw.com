# What's New?
- Pixel search bar replacement [EXPERIMENTAL]
- Intents - For external app controls (WIP)
- Travel compass in are we there yet?
- Per-widget screen off configuration for double tap
- Dedicated flashlight settings
- Flashlight pulse disable during DND
- Shut up restore notification option
- Shizuku improvements (Token entry, Missing permission notification ... )
- DNS tile improvements (Can disable auto mode)
- DIY improvements
   - New Triggers
      - Action Shortcut
      - Pixel search bar tap
   - New Actions
      - Screen off
      - Media controls
      - Assistant
      - Take screenshot
      - Toggle media volume
      - Like current song
      - Circle to search
      - Pin foreground app (very broky)
- Action shortcut
   - This is especially useful for Pixel users as now you can map this shortcut as your double tap gesture on back to trigger anything from DIY automations! (My fav: like current song)
   - Will only show up in your app drawer if you have an automation set for it
- Ultra HDR support for watermark
- [WearOS Essentials improvements](https://github.com/sameerasw/essentials-wear/releases/tag/v3.0)
   - Better sync
   - Remote wireless debugging toggle
   - Sync sound mode between watch and Android
   - More controls (Flashlight pulse, AOD, tap to wake)
   - Control customization and ordering
   - Are we there yet on device info tile
   > Keep in mind that at the moment, you need to grant the permissions via ADB. I will look for an easier method)
- Swipe tabs are back! and better! (Can be turned off)
- Search fixes
- and a lot more fixes and improvements
