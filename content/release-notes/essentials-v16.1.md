## What's New?
- ### Better updates!
   - From the next release, Essentials will be able to download and prompt you to install the updates better and easier!
   - More reliable background update checks
   - Some minor UI indicators for available udpates
   
- ### In-app translator!
   - Crowdin translations will be disabled soon due to we having tooooo much words for the free plan
   - Instead of that, now, you can enable the "Translate mode" in settings and help translating texts inside the app easily
   - Currently many cards, sliders, and help text components are supported but the support will be expanded to more text in the future
   - Long press any UI element with text to see if it's translatable in translate mode
   - If so, you can edit the string to show in your language and submit multiple edits altogether
   - Contributions will be made with your signed-in GitHub accounts and a PR will be opened with changes
   - I can not guarantee to approve all contributions as there will be conflicts as many users opening PRs, i will look for solutions for this.
   - And much much love for the community (‘∀’●)♡
   
- ### Wallpaper checking was updated
   - Instead of twice a day, now it will check every day at the time you enabled auto updates and re-try after 30mins a couple of times if no new wallpaper was found.
