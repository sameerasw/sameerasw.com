# What's New?
- Wallpaper
   - Curated daily wallpaper from my unsplash collection
   - Auto update wallpaper
   - Merge curated wallpapers and live wallpapers together into one single feature
   - Clear all custom videos in live wallpaper option
   - Many more wallpaper fixes
- Pocket mode by @Mudit200408 
   - Pocket detection to turn off screen (Disabled on Pixels by default - Existing feature)
   - Pocket detection for flashlight to auto turn off
- OEM specific feature limits
   - Some Pixel exclusive features such as lock screen clock, charging mode and a few were removed from others
   - Pocket mode is unavailable on Pixels by default
   - You can enable all features in Settings, Proceed with caution
- More launcher shortcuts
   - Wallpaper was added
   - Your favorite 2,3 entries will show up below existing shortcuts
- Smoother flashlight pulse on lower brightness settings
- New developer option was added (for me 👉👈) to refresh daily wallpaper for Essentials and New tab extension because why not?
- Major code refactor and optimization
- And a few bits and pieces of fixes and improvements here and there
