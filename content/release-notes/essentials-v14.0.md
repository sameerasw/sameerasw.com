# What's New?
- [WearOS Android controls and sync](https://github.com/sameerasw/essentials-wear/releases/tag/v2.0)
   - Lock your Android remotely
   - Control flashlight and brightness
   - Toggle Android sound mode
   - A lot more future plans for a closer experience with both devices
- Shut Up! I know what I'm doing
   - Is something we tell the apps which blocks itself from the user if they have developer options, debugging or even accessibility services are enabled and this feature is made to say F U t(-_-t) to them. (experimental) 
- Pixel lock screen clock customizations
   - Mostly the same clock styles but the dynamic clock getting more variable font controls
   - With future plans of integrating automations
- Screen rotation suggestion button toggle
- Pause trips for Are we there yet?
- More marquee text for larger scale UI
- Shortcut app icon performance improvements
- More fixes and improvements
