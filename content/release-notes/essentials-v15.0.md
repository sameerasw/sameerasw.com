# What's New?
- Essentials on Display
   - Custom AoD implementation from the old Ambient glance feature with some more customizations and options,
   - Clock styling
   - More media album art customizations
   - Unread notification badges (experimental)
   - Improved media detection
   - Updated screensaver with unified customizations
- Feature animations with lottie
- Removed GSMArena powered device specs **
- Migrated the App management to the Your Android feature
- Improved search
   - Better UI
   - Search history
- App freezing launcher entry (You can turn off)
- Visual improvements with more variable fonts
- Battery widget improvements with Jetpack Glance
- Major code refactor and optimization
- Target Android 17
- Upgraded the compiled Java v11 -> v21
- Migrated media to webp (Trims a lot of file sizes)
- Fix battery widget theme adaptivity
- & more improvements and fixes

> ** Why device specs were removed?
>    The feature was powered by the GSMArena website which does not offer an API. So we had to rely on a web scraper which did not provide accurate information for certain devices and especially for devices with similar specs and model numbers. Add to that, this component did not add any valuable user facing-features hence it was removed.
