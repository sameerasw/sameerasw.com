# What’s new?

* Fix menubar pop-up focus
* Notification silence button
* Improve network stability
* Improve auto re-connect
* New Native Mirroring Experience \[Experimental\]
* Copy ADB logs
* Keyboard navigation for the file browser
* Translations
* Improved UI blur and shades
* App UI scaling fixes
* Share sheet entry for text
* Clipboard and connection pill
* Remote layout switch button
* Service manager for improved background service management
* Dynamic app shortcuts
* And a lot more fixes and improvements
