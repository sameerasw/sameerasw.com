# Why 3.0?

It's a big update with a lot of surface and also behind the scene upgrades and fine tuning with new features as well. Also thanks to [Syntrop2k2](http://t.me/Syntrop2k2) some new icons and branding was introduced to the app to unify on both platforms.

# What's the deal with QuickShare?

It was requested a lot and I was also frustrated with the previous half broken file share implementation so replaced it with QuickShare support which made this my primary file transfer app from Blip. Months of research, trial and error and finally I was able to get it working!

There are some limitations tho, Just like other QuickDrop and NearDrop apps, both devices needs to be on the same local network to be discoverable but at least I was able to implement automatic file accepting to make the experience better.

Hopefully Google will expand AirDrop support on Android to more devices but until then, Hope I helped at least one of you :D
