# What's new?
- Glove mode (+ QS tile)
- Enhanced lock screen security with QS disabled
- Advanced statusbar customizations
- Hide gesture bar
- New Pixel notification lighting effects
- Flashlight pulse brightness intensity
- Random shapes in ambient glance toggle
- SystemUI force restart
- New statusbarManager for handling all related shell commands
- Improved QS tile activation logics
- And more improvements and fixes
