# What's New?
- An attempt at fixing the overflowing bottom toolbar
- A lot of Material 3 Expressive UI component updates instead of custom components.
- Your Android
   - Diagnostics and Device check for Pixel devices (Battery & storage health + hardware & sensor testing)
   - Fix crash for some devices
   - > Note: device detection might still have misses, Will try to fix soon as it's mostly due to GSMArena discovery
 - App freezing
   - You can now search apps
   - Added app context menu (long press) to unfreeze or even remove easily
   - Add shortcut to home in long press menu
   - Made the dedicated frozen apps activity the same as the freeze tab UI
- Are we there yet
   - Full redesign
   - Ability to save destinations
   - Customize destination details
   - ETA
- More improvements and fixes
